Creating a footer

  • mac
  • windows
  • linux
  • all

Wikis can have footers that contain additional information, such as copyright dates, creative licensing, or information on contributing.

  1. In the upper-right corner of any page, click your profile photo.
  2. On your profile page, click the Repositories tab, then click the name of your repository.
  3. In your repository's right sidebar, click Wiki.
  4. At the bottom of the page, click Add a custom footer.
  5. A custom footer can contain any markup that GitHub supports. The default choice is Markdown, but you can use the "Edit mode" drop-down menu to switch to a different markup language.
  6. Use the text editor to type the content you want your footer to have. You can also use the wiki toolbar at the top for entering text via a graphical WYSIWYG editor.
  7. Enter a commit message describing the footer youâ€™re adding.
  8. To commit your changes to the wiki, click Save Page.
