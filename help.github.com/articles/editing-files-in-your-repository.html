Editing files in your repository

  • mac
  • windows
  • linux
  • all

You can edit files directly on GitHub in any of your repositories using the Ace editor.

  1. In your repository, browse to the file you want to edit.
  2. In the upper right corner of the file view, click to open the file editor.
  3. Make any changes you need to the file.
  4. Click Preview changes to see how your changes will look once they are rendered.
  5. At the bottom of the page, under "Commit changes", type a short, meaningful commit message.
  6. Click Commit changes.
