Making a public repository private

  • mac
  • windows
  • linux
  • all

If you have a paid account, you can convert your public repositories to private ones.

Tip: If you are on the free plan, or have no private repositories available, you will need to upgrade before you can switch the repository to private.

  1. Open the repository's page on GitHub.
  2. Click Settings.
  3. Click Make private.
  4. Read the warnings.
  5. Seriously, read the warnings!
  6. Type the name of the repository that you want to make private.
  7. Click I understand, make this repository private.

Warning: When a public repository is made private, public forks of the public repository are detached and put into a new network. They are not made private. For more information, see What happens to forks when a repository changes visibility.
