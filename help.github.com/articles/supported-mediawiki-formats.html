Supported MediaWiki formats

  • mac
  • windows
  • linux
  • all

Wikis support content that's written just like everywhere else on GitHub. We also support some of the syntax supported by MediaWiki.

No matter which markup language your wiki page is written in, certain MediaWiki syntax will always be available to you. For security and performance reasons, some syntaxes are unsupported.

Supported MediaWiki syntaxes

  • Links (except Asciidoc)
  • Horizontal rules via ---
  • Shorthand symbol entities (such as &delta; or &euro;)

Unsupported MediaWiki syntaxes

  • Transclusion
  • Definition lists
  • Indentation
  • Table of contents
  • Contact a human
