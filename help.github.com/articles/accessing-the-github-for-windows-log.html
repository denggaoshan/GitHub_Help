Accessing the GitHub for Windows log

  • mac
  • windows
  • linux
  • all

When reporting an issue with GitHub for Windows, it's super helpful to include GitHub for Windows's log file. Here's how to get it:

  1. Reproduce the bug
  2. Quit the app
  3. Hold the Windows key and press R
    • On Windows Vista or later, type %LOCALAPPDATA%\GitHub
    • On Windows XP, type %USERPROFILE%\Local Settings\Application Data\GitHub. Note that this path may be localized if you're using a non-English version of Windows.
  5. Press Enter
  6. Copy TheLog.txt from the window that appears
  • Contact a human
